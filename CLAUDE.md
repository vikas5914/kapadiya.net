# kapadiya.net

Personal site of Vikas Kapadiya (AI engineer at Shopos AI). Astro 7 static site with Tailwind 4, deployed at https://kapadiya.net.

## Design Context

Strategic and visual context for any UI work lives in two root files — read them before designing or changing any surface:

- **PRODUCT.md** — register (brand), platform (web), audience (engineers/OSS community), positioning ("AI engineer who ships"), brand personality (precise, understated, technical), anti-references, and design principles.
- **DESIGN.md** — the "Machine Shop" design system: dark industrial palette (Shop Floor #141414, Safety Orange #f25c05 accent), Geist/Geist Mono typography, zero border radius, hairline-border grids, flat elevation. Machine-readable tokens in its frontmatter; `.impeccable/design.json` carries the extended tokens and component snippets.

Stay inside these constraints: one orange accent, square corners, mono uppercase labels, no new shadows, reduced-motion alternatives for all animation.
